1. Instdall Node.js
2. Install Crestron utilities CLI

   ```jsx
   sudo npm install -g @crestron/ch5-utilities-cli
   sudo npm install -g @crestron/ch5-shell-utilities-cli
   ```

3. Install VS Code
4. Install CH5 VS Code Extension
